ive been experimenting lately with some unique 90s style 3D shading, primatives, lighting, and landscapes,,,

as you can see it has led to some quite interesting results !

all of the above scene was rendered inside of blender using cycles engine renderer, and then dithered further in post processing within shotcut

whilst this particular scene is just an experiment, i have plans to implement 90s style videos into my game i am developing

i was heavily inspried by games such as fallout, starcraft, sid meier's alpha centauri, resident evil, warcraft 2, and many many more games from my childhood

as you can see these are a LOT more detailed than my haphazardly thrown together scene, but you can see the vision im going for here - with time and practice this is what i aim to acomplish, but at the least the scene setup (render settings, postprocessing, etc.) is all configured to be as close to 90s style as i can make it without using actual 90s era software

well thats about it for today's blog, i hope u enjoyed and ill be sure to update my site with more information about my recent development projects - bai ^~,
